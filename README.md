# Conda环境

1. 创建 Conda 环境：

   ```bash
   conda create --name mnist_env python=3.6 
   ```
2. 激活环境：

   ```bash
   conda activate mnist_env
   ```
3. 安装其他依赖：

   ```bash
   conda install tensorflow=1.12 numpy=1.15 matplotlib=3.0
   ```
